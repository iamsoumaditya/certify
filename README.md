# 🎓 Certify — Because effort is overrated.

> **Generate beautiful, completely unofficial Udemy-style certificates for literally any course you claim to have completed. No studying required.**

Certify is a fun, side-project web app where you can create pixel-perfect fake Udemy certificates in about 30 seconds. Fill in a course name, hit generate, and boom — you are now a certified expert in whatever you typed. Share the link. Put it on LinkedIn. Frame it. We dare you.

## 🚀 How to Generate a Certificate

1. **Sign up or log in** at [getcertified.tech](https://getcertified.tech) (or run it locally — see below).
2. Click **"Get Certified →"** or go to the **Dashboard**.
3. Fill in the form:
   | Field | Example |
   |---|---|
   | Student Name | Your name (auto-filled from your account) |
   | Course Name | Advanced Napping Techniques |
   | Instructor Name | Dr. Sleep |
   | Completion Date | Any date you like |
   | Total Hours | 10.5 total hours |
4. Click **"Generate Certificate"** and wait ~5 seconds.
5. You'll be redirected to your shiny new certificate page. 🎉

---

## 📥 Downloading Your Certificate

Once your certificate is generated, you'll land on its dedicated page. As the owner you'll see two download buttons:

- **Download JPG** — saves a high-resolution image of the certificate.
- **Download PDF** — saves a print-ready landscape PDF.

Every certificate also has a permanent, shareable URL in the format:
```
getcertified.tech/certificate/<uuid>
```
Bookmark it, share it, or revisit it at any time.

---

## 🔁 Retrieving Past Certificates

Already generated a certificate and want it back? No problem:

1. **Sign in** with the same account you used originally.
2. Your certificates are saved to your account in our database.
3. Navigate to the certificate URL (e.g. from your browser history) or head to the **Dashboard** — all your past certificates are linked there.
4. Download them again in JPG or PDF whenever you need.

## 🛠️ Running Locally

This is a [Next.js](https://nextjs.org) app. To run it on your own machine:

### Prerequisites

- Node.js 18+
- A [Clerk](https://clerk.dev) account (for auth)
- A [Cloudinary](https://cloudinary.com) account (for image storage)
- A PostgreSQL database (e.g. [Neon](https://neon.tech))

### Setup

```bash
# Clone the repo
git clone https://github.com/iamsoumaditya/certify.git
cd certify

# Install dependencies
npm install

# Copy environment variables and fill in your keys
cp .env.example .env.local
```

Add the following to your `.env.local`:

```env
# Clerk
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=...
CLERK_SECRET_KEY=...

# Database (PostgreSQL)
DATABASE_URL=...

# Cloudinary
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
```

```bash
# Push the database schema
npx drizzle-kit push

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and enjoy.
